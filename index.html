About

I'm a digital + UX enthusiast living in Madison, Wisconsin. My academic background is in strategic communication, but I fell into the digital world through work and classes in human-computer interaction, typography and interactive cartography. My designs focus on minimalism, and my process is extremely iterative and collaborative to reach that point of prime simplicity.

Portfolio


SHHH

Created for a human-computer interaction course, shhh is an app designed to help students find spots to study based on their preferences. Since this selection by preferences is the app's unique selling point, I created a prototype that put this selection process at the forefront.


Devil's Lake State Park

An interactive slippy map of Devil's Lake State Park near Baraboo, Wisconsin, created using the JavaScript library Leaflet for a final group project in an upper-level interactive cartography course.My contribution was basic UI/UX design, iconography and color and typography choices.


Chicago's Poor Health

An interactive map created using the JavaScript library d3 and TopoJSON showing the correlation between poverty and cause of death in Chicago for an upper-level interactive cartography course.


The New Commute

An interactive map created using the JavaScript libraries Leaflet and jQuery AJAX showing the increase in bike commuting in the U.S. over time for an upper-level interactive cartography course.


Didot Cards

Created for a course in typography and communication design, this deck of trading cards shows each glyph of Didot, a French modernist serif. Since Didot is often used in fashion magazines and high-society posters and brochures, I decided to focus the desk on fashion, with each glyph's dominant anatomical feature representing an item of clothing.


First Things First Layouts

In a typography and communication design course, we were assigned to create three layouts for the First Things First 2000 manifesto, each respectively representing the Japanese priciples of formality, semi-formality and informality.


San Francisco Rent

An infographic showing the impact of San Francisco's high rental rates.


Chicago Crime

A choropleth and proportional symbol map representing both violent and overall crime in Chicago.
