About

I'm a writer and designer living in Madison, Wisconsin. My academic background is in journalism and strategic communication, but I have a passion for visual and digital communication.

Design


Devil's Lake State Park

An interactive slippy map of Devil's Lake State Park near Baraboo, Wisconsin, created using the JavaScript library Leaflet for a final group project in an upper-level interactive cartography course. My primary role was creating the branding, spec sheet and icongraphy.


Chicago's Poor Health

An interactive map created using the JavaScript library d3 and TopoJSON showing the correlation between poverty and cause of death in Chicago for an upper-level interactive cartography course.


The New Commute

An interactive map created using the JavaScript libraries Leaflet and jQuery AJAX showing the increase in bike commuting in the U.S. over time for an upper-level interactive cartography course.


San Francisco Rent

An infographic showing the impact of San Francisco's high rental rates.


Chicago Crime

A choropleth and proportional symbol map representing both violent and overall crime in Chicago.


Europe

As a student in an introductory cartography class, one of our first projects was creating a reference map of Europe. It includes over 100 labels of countries, major cities and bodies of water.

Writing

I served as editor of the 2014 edition, where I finalized the editorial philosophy and facilitated story assignments and development. I also wrote a feature piece and conducted a Q&A with Chef Tory Miller.

The Daily Cardinal

The Daily Cardinal is UW-Madison's only daily newspaper. I wrote for the arts section for two semesters.
